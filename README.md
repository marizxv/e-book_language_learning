# Aplikacja mobilna do nauki języka obcego poprzez czytanie e-booków

## Opis projektu

Projekt zakłada stworzenie aplikacji mobilnej wspierającej naukę języków obcych poprzez czytanie własnych e-booków w języku obcym. Aplikacja umożliwia użytkownikowi wygodne czytanie książek, tłumaczenie nieznanych słów oraz budowanie własnej bazy słownictwa do późniejszych powtórek.

System analizuje również poziom trudności tekstu oraz dopasowanie książki do umiejętności językowych użytkownika. Dodatkowo aplikacja ocenia przydatność poznawanych słów, pomagając skupić się na słownictwie najczęściej wykorzystywanym w praktyce.

Projekt łączy funkcje:

* czytnika e-booków,
* tłumacza,
* systemu powtórek słownictwa,
* inteligentnego wsparcia nauki języka.

# Mobile Application for Learning Foreign Languages Through Reading E-Books

## Project Description

The project aims to develop a mobile application that supports foreign language learning through reading personal e-books in a foreign language. The application allows users to upload books, translate unknown words, and build their own vocabulary database for later revision.

The system also analyzes the difficulty level of the text and evaluates how well the selected book matches the user’s language skills. Additionally, the application assesses the usefulness of newly learned words, helping users focus on vocabulary that is most practical in everyday use.

The project combines:

* an e-book reader,
* a translation tool,
* a vocabulary revision system,
* intelligent language-learning support.

---

## Main Features

* Upload and read personal e-books
* Instant translation of selected words
* Personal vocabulary collection
* Vocabulary revision and repetition
* Text difficulty analysis
* Book-to-user skill matching
* Useful vocabulary prioritization
* Personalized learning support
